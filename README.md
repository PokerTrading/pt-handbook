## 🎯 这套策略解决什么问题

大部分在市场里待过几年的人，看图表的能力都不差。但结果呢？要么看到了没敢做，要么做了但仓位不对，要么进去了但拿不住。

问题出在分析之后的所有环节——什么时候出手、下多大仓位、止损放哪里、什么时候该走。

Poker Trading 用**拍卖理论**读结构，用**德州扑克**管执行。从打开图表到平仓下桌，每一步都有规则、每一步都有依据。

## 🔑 核心概念速览

| 概念 | 一句话解释 |
|------|-----------|
| 市场 | 所有人同时参与的双向拍卖 |
| 价格 vs 成交量 | 价格是广告，成交量是投票 |
| 价值区（VA） | 70%成交量的价格区间 = 大多数人觉得合理的范围 |
| 一级边界 | 8个发牌位：PH/PL/VAH/VAL/IBH/IBL + Excess High/Low |
| 信号 | 只有两种：接受（AC）和 拒绝（RE） |
| 桌风 | VA的迁移方向 = 顺风优先 |
| 验牌 | 确认信号是真的：方向→速度→停留/能量 |
| 牌力 | AA/KK/AK/AQ 四档 = 共振 × 信号强度 |
| 买定离手 | 入场前做完功课，入场后按计划执行 |

## 🛠️ 三个核心工具

| 工具 | 作用 | 输出 |
|------|------|------|
| **Poker Table** | 画牌桌地图 | VA · POC · 关键位 · Excess/Poor |
| **Poker Flow** | 读桌风方向 | 多头/空头/回调/平衡 |
| **Poker Radar** | 捕信号+评分 | uAC/dAC/uRE/dRE + 1-10分 |

## 🚀 新手起步建议

1. **读完 Ch1-Ch4**，理解市场、工具、桌风、信号
2. **用双屏布局**（背景桌 + 执行桌），偏见 = 执行桌桌风方向
3. **只打顺势牌的 AA 和 KK**，其余一律弃牌
4. **前100笔用 AQ 级仓位**（每笔0.8%风险），重点看规则执行率
5. **每笔交易填复盘表**（Ch9模板），3分钟投资，一个月后回报巨大
