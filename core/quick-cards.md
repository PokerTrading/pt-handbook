## 卡4 · 验牌链路

```
信号出现 → Excess? → 是: 跳过①直接到②
                    → 否: 走①
           → ① 方向/评分
              RE: 有反弹? (No→弃牌)
              AC: 评分7+? (Yes→直接过)
           → ② 响应速度 (RE和Excess)
              ≤3根K线 + 远离 + 实体为主 → 直接过
           → ③ 停留 OR ④ 能量
              ③ 待在新方向≥2根K线 → 过
              ④ 同向新信号 → 过
              都没有 → 弃牌
```

**优先级：** 🟢A(7+/双Excess)立刻验 · 🟡B(4-6/单Excess)观察 · 🔴C(<4)跳过

## 卡9 · 入场检查清单（Ch4→Ch7一条线）

```
□ Ch4 发牌：8个一级边界出事了？（VA内=噪音→跳过）
  □ 识别来源：Table画出全部8个边界 · Radar在6个边界上捕捉AC/RE + Table识别2个Excess · 共同存在以Excess为准
  □ Radar验牌优先级：A档(7+)立刻验 · B档(4-6)观察 · C档(<4)跳过
  □ Excess验牌：跳过评分，从②速度开始
  □ 验牌通过？（①方向→②速度→③停留/④能量）
    ↓ 通过
□ Ch5 牌型：顺风/逆风/诈唬/延续？
  □ Radar牌力：共振×信号强度 → AA/KK/AK/AQ
  □ Excess牌力：单Excess(KK起步) / 双Excess(AA起步) × 共振 · 逆风降一级
  □ 纹理：干燥/中性/湿润？
  □ 降级：桌风分歧?/纹理?/不确定? → 最终牌力
  □ 决策总表查表 → 🟢Go / 🔴弃牌
    ↓ Go
□ Ch6 甜点位：结构锚点>K线开盘价>Fib回撤 · 够不到→弃牌
  □ SL：结构SL（首选）或数学SL（降级）· ≥最小下限
  □ TP1：第一个结构阻挡
  □ R:R：≥门槛？ · 不达标→弃牌
    ↓ 达标
□ Ch7 仓位：牌力→手数 · 前置or分街 · 安全/价值/贪婪分配 · 挂单入场
```
