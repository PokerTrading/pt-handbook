## 卡4 · 验牌链路

```
信号出现 → ① 方向
             RE: 有反弹? AC: 继续走? Excess: 远离?
             No → 弃牌
          → ② 时间
             ≥3根K线待在新方向不回去
             回去了 → 弃牌
          → ③ 能量
             同向AC且不被Radar否定?
             有 → 先手入场（直接挂甜点位）
             没有 → 后手入场（等回踩不破再挂）

快速通道: 去程中同向AC且不被Radar否定 = ①②③同时过 → 先手
```

**优先级：** 🟢A(7+/双Excess)立刻验 · 🟡B(4-6/单Excess)观察 · 🔴C(<4)跳过

## 卡6 · 风控速查

**三层资金：** 牌本（Bankroll）→ 子弹（Buy-in）→ 手数（Hands）

| 规则 | 内容 |
|------|------|
| 牌本 | 全亏不影响生活 |
| 子弹 | ≥33手（扛10笔AQ+缓冲）|
| 单笔上限 | ≤12手（AA天然封顶）|
| 总持仓 | ≤24手 |
| 日熔断 | 日止损≥2笔停 |
| 周熔断 | 周止损≥4笔停+复盘 |
| 子弹≥60% | 正常打 |
| 子弹40-60% | 牌力降一级 |
| 子弹20-40% | 只打AA（降一级9手）|
| 子弹<20% | 停止当颗 |
| 连续2颗打完 | 休息一周+复盘 |

## 卡9 · 入场检查清单（Ch4→Ch7一条线）

```
□ Ch4 发牌：8个一级边界出事了？（VA内=噪音→跳过）
  □ 识别来源：Table画出VAH/VAL+识别Excess(始终可见) · Radar标记PH/PL/IBH/IBL(事件触发) · Radar在6个边界上捕AC/RE · 共同存在以Excess为准
  □ Radar验牌优先级：A档(7+)立刻验 · B档(4-6)观察 · C档(<4)跳过
  □ 验牌通过？（①方向→②时间→③能量→先手/后手）
    ↓ 通过
□ Ch5 牌型：顺风/逆风/诈唬/延续？
  □ Radar牌力：共振×信号强度 → AA/KK/AK/AQ
  □ Excess牌力：单Excess(KK起步) / 双Excess(AA起步) × 共振 · 逆风降一级
  □ 纹理：干燥/中性/湿润？
  □ 降级：桌风分歧?/纹理?/不确定? → 最终牌力
  □ 决策总表查表 → 🟢Go / 🔴弃牌
    ↓ Go
□ Ch6 甜点位：结构锚点>K线开盘价>Fib回撤 · 够不到→弃牌
  □ SL：结构SL（首选）或数学SL（降级）· ≥最小下限
  □ TP1：第一个结构阻挡
  □ R:R：≥门槛？ · 不达标→弃牌
    ↓ 达标
□ Ch7 仓位：牌力→手数 · 前置or分街 · 安全/价值/贪婪分配 · 挂单入场
```
