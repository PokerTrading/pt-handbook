# Poker Trading一页纸总览

## 核心世界观
市场 = 所有人同时参与的双向拍卖。价格是广告，成交量是投票。
不预测涨跌，只识别结构——价值在哪里、价格偏离了多远、市场正在接受还是拒绝。
拍卖市场的结构分析能力 + 德州扑克的执行框架 = Poker Trading。

## 四张牌桌（四个时间级别）
| 牌桌 | VP计算范围 | K线 | 持仓周期 | 桌风回溯 | IB窗口 | 看盘频率 |
|------|--------|-----|---------|---------|--------|---------|
| 波段桌 | 1月VP | 日线 | 数周～数月 | 月/3 | 月初前3天 | 每天收盘后1次 |
| 短线桌 | 1周VP | 4H | 数天～两周 | 周/4 | 周一全天 | 每4H或一天2-3次 |
| 日内桌 | 1日VP | 30m（默认）/ 15m（可选） | 24小时内 | 日/5 | 开盘后1小时 | 每30-60分钟 |
| 分时段桌 | 1时段VP | 5m | 数小时 | — | 时段前15-20分钟 | 盘中实时 |

更高级别VP（季度VP、年VP）用作偏见桌和背景桌，不作为执行桌。
加密货币VP周期以UTC时间为界（UTC周一0:00=新周，每月1日0:00=新月）。

看盘布局：双屏（新手路线）背景桌+执行桌，偏见=执行桌桌风方向；三屏（进阶）加偏见桌。背景桌主要提供共振参考，偏见桌提供偏见方向（进阶）。
一次只坐一张桌。

## 牌桌切换规则
当前VP未成熟 → 临时用上一周期VP为主桌。
当前VP成熟（清晰稳定的POC+VAH/VAL）→ 切回当前桌。
最迟切换时限：月桌第10天 / 周桌周三 / 日桌第8小时。

## 桌风与偏见（Table Flow & Bias）
核心原则：**顺风优先**——多头桌风优先做多，空头桌风优先做空（不是绝对，但大大提高胜率）。
偏见决定你优先盯哪些边界。基础：偏见=桌风方向。进阶：偏见=偏见桌未完成的牌方向（覆盖桌风）。
偏见桌有未完成的牌（一级边界/POC上的AC或RE，或Table识别的Excess，未被反向AC否定）→ 偏见=那手牌方向。偏见桌没牌 → 看背景桌。都没有 → 偏见=桌风方向。
偏见与桌风一致=最理想（顺势牌高置信）。冲突=偏见定注意力，桌风定牌型。
偏见不替代验牌、牌力评估、下注决策。

## 决策链（8章I/O链路）
```
Ch3 桌风方向 + 成熟VP + 建立偏见（偏见桌有未完成的牌→偏见=牌方向，否则→偏见=桌风）
 ↓
Ch4 在偏见方向边界等发牌 → 验牌（确认事件是真的）→ 验牌通过的信号
 ↓
Ch5 牌型（哪种机会）→ 牌力（多好）→ 纹理（环境配吗）→ 降级 → 最终牌力
 ↓
Ch6 甜点位 → SL → TP → R:R → Go/No-go
 ↓
Ch7 仓位 → 下注模式 → 四街执行 → 买定离手
 ↓
Ch8 风控 · 情绪 · 长期生存
```

## 边缘交易原则
只因边界事件开仓——发牌必须来自8个一级边界，入场价位在甜点位上。VA内部的信号一律当噪音忽略。
一级边界（全部由Table画出）：Radar在 PH / PL（只RE）/ VAH / VAL / IBH / IBL（AC和RE）共6个边界上捕捉K线级AC/RE事件 + Table在VP级别识别 Excess High / Excess Low 共2个结构边界。
发牌两种检测方式（同级并列）：Radar在K线级别实时捕捉AC/RE事件，Table在VP级别识别Excess结构演化。PH/PL和Excess都出现在周期高低点区域，共同存在时以Excess为准。Excess验牌跳过评分判断，从响应速度开始。
POC不是发牌位——它是磁铁/路况参考，在以下场景有辅助价值：
- 验牌能量：边界信号后POC处的同向AC可作验牌证据
- 共振加分：执行桌边界与偏见桌/背景桌关键位（含POC）重合时提升牌力
- 方向参考：POC持续迁移辅助判断方向
- 翻转入场锚点：边界事件后价格穿越POC（AC确认），POC角色翻转（支撑↔阻力），可作回踩/回抽甜点位。前提：先有边界事件+穿越必须AC确认+回踩时入场。同一VP周期内有效。
否定锚点穿越定义：执行桌K线收盘在锚点外侧才算穿越，影线碰到不算。

## 验牌优先级
A档（立刻验牌）：Radar 7+ → 立即走验牌链路
B档（耐心观察）：Radar 4-6 → 观察后续
C档（暂时跳过）：Radar 4以下 → 忽略
验牌在当前牌桌K线上进行。验牌不限时间，唯一过期条件=VP更新后边界位移。

## 验牌（入场和加注通用）

| | RE验牌 | AC验牌 |
|--|--------|--------|
| **①** | 方向：有反弹？无→弃牌 | 评分：7+→直接过 |
| **②** | 速度：≤3根K线+远离+实体为主→直接过 | （跳过，直接看③④） |
| **③** | 停留：≥2根K线待在新方向 | 站稳：不回内侧 |
| **④** | 能量：同向AC出现 | 连续AC推进 |
| **失败** | 回到拒绝位 | 回到内侧→假突破 |

①通过但②③④未明确=验牌进行中，继续观察不入场。
能量信号有效期=当前VP周期内。验牌过期=VP更新后边界位移（过期≠失败）。

## 三种入场逻辑（边界优先）
| 入场逻辑 | 边界 | 事件 | 结果 |
|---------|------|------|------|
| 买便宜/卖贵 | 下方边界RE / 上方边界RE | 拒绝弹回 | 在支撑/阻力位入场 |
| 追真突破/跌破 | AC成功+验牌通过 | 突破站稳，新价值区 | 回踩突破位入场 |
| 假突破反做 | AC失败，价格回到原侧 | 突破失败 | 在原边界反方向入场 |

## 五种牌型（交易方向 vs 桌风方向）
| 牌型 | 定义 | 门槛 |
|------|------|------|
| 顺势牌 | 交易方向=桌风+VA迁移方向 | 标准（主食） |
| 反转牌 | 交易方向逆桌风+VA迁移 | AA降级入池；KK条件入池（7+ + 纹理不湿润 + R:R≥2:1，三条件缺一弃牌） |
| 诈唬牌 | 假突破后反做（顺势诈唬/反转诈唬） | 按方向归类：顺风=顺势门槛，逆风=反转门槛 |
| 中性牌 | 平衡桌风下的任何信号 | AA标准入池，KK轻仓，AK多数弃牌（仅7+时轻仓），AQ弃牌，R:R≥1.8:1 |
| 延续牌 | 错过原始入场后的小平衡补救 | 轻仓3-4手，三前提缺一不做 |

平衡桌风判定：回溯窗口内VA移动同向<60%=平衡。≥60%同向=回调（方向已确立，近期暂停）。

## 四种牌力（最终牌力 = 原始牌力 - 降级）

**Radar发牌：** 信号强 = Radar 7+。信号不强 = Radar 4-6。

| 牌力 | 共振 | 信号强度 |
|------|------|---------|
| AA | 有共振 | 信号强 |
| KK | 有共振 | 信号不强 |
| AK | 无共振 | 信号强 |
| AQ | 无共振 | 信号不强 |

**Excess发牌（Table识别的2个一级边界）：** 牌力由结构强度×共振决定，无Radar评分。

| 牌力 | 单Excess | 双Excess |
|------|---------|---------|
| 有共振·顺势/中性 | KK | AA |
| 有共振·反转 | AK | KK |
| 无共振·顺势/中性 | AK | KK |
| 无共振·反转 | AQ | AK |

双Excess反转牌免7+条件直入池（方向穷尽的结构证据）。纹理和R:R照常走。

共振量化：两个关键位价差≤执行桌VA宽度10%=共振。跨周期距离越大越有价值（偏见桌>背景桌）。
降级来源：桌风分歧/判断不确定 → 降一级；湿润纹理 → 降一级；极端湿润 → 直接弃牌。
牌力降级不叠加，取最严格。但多个降级来源同时存在→仓位额外减半。降级后低于AQ → 弃牌。

## 牌面纹理（三档）
| 纹理 | 特征 | 对牌力的影响 |
|------|------|-------------|
| 干燥 | 路上全是绿色/白色通道（LVN），无阻碍 | 不降级 |
| 中性 | 有红色块（HVN）但不密集，路有阻碍但可通过 | 不降级，但注意TP可能遇阻 |
| 湿润 | 红色块密集或面积大，路上障碍重重 | 降一级；极端湿润 → 直接弃牌 |

## 甜点位（RE和AC分开）
RE甜点位：🥇结构锚点（IBH/IBL、VAH/VAL、POC）🥈信号K线开盘价 🥉斐波那契0.618/0.786
AC甜点位：🥇突破位本身（回踩入场）🥈POC翻转位 🥉斐波那契0.618/0.786
"够不到"=甜点位与当前价格距离≥信号K线振幅×1.5。三层都够不到=不入场。

## 止损（SL）
从甜点位出发往反方向找最近结构位 → SL画在外侧。
SL可用结构位：一级边界 + HVN边缘 + POC。LVN不算。
最小SL下限：黄金分时段≥$3/日内≥$5/短线≥$10/波段≥$15。BTC分时段≥$100/日内≥$200/短线≥$500/波段≥$1000。低于下限放弃本局。
数学SL（降级方案）：仅限AA/KK+Radar 7+。仓位降一级+R:R硬要求+0.5。AK/AQ不允许数学SL。Excess发牌无Radar评分，不适用数学SL。

## 目标位（TP）
TP1=甜点位到交易方向第一个结构阻挡（HVN边缘/下一个一级边界/对面VA边界）。
TP2=第二个结构阻挡或偏见桌/背景桌关键位。
TP3=偏见方向极端目标（偏见桌PH/PL）或跟着走到结构反转。

## 盈亏比（R:R）
R:R = 甜点位到TP1 ÷ 甜点位到SL
顺势牌（含顺势诈唬）：≥ 1.5:1
中性牌（平衡桌风）：≥ 1.8:1
反转牌（含反转诈唬）：≥ 2:1
数学SL情况：在对应牌型基础上+0.5
用TP1算，不用TP2/TP3（保守设计，确保仅到TP1即正）。不达标弃牌。

## 仓位与下注
| 牌力 | 最大手数 |
|------|---------|
| AA | 10手 |
| KK | 8手 |
| AK | 6手 |
| AQ | 4手 |

两种下注模式（一局只用一种，不确定时用前置）：
- **前置下注**：计划手数一次入场，后面只做止盈+SL管理
- **分街下注**：初始轻仓，等结构事件加注（最多两次），每次加注当新牌处理
反转牌只用前置。

仓位三组：安全组40%（TP1平仓）+ 价值组40%（TP2平仓）+ 贪婪组20%（TP3/结构反转/对侧Excess下桌）。纹理湿润→50/40/10，纹理干燥+AA→30/40/30，也可按手数均分或自定义。

## 四街打法
| 街 | 前置下注 | 分街下注 |
|----|---------|---------|
| 翻牌前 | 计划手数入场 | 初始轻仓入场 |
| 翻牌 | TP1→安全组平仓+价值组/贪婪组SL移入场价 | 第一次加注（路上新结构确认时） |
| 转牌 | TP2→价值组平仓+贪婪组SL移TP1 | 第二次加注（第一个主要结构位回踩时） |
| 河牌 | TP3/结构反转/对侧Excess→贪婪组下桌 | 止盈阶段（安全组TP1/价值组TP2/贪婪组TP3+） |

SL管理：TP达成前不碰SL → TP1后安全组平仓，价值组+贪婪组SL移入场价 → TP2后价值组平仓，贪婪组SL移TP1 → 贪婪组等TP3/结构反转/对侧Excess下桌。

## 买定离手原则
"买定"：入场前做完所有功课，不轻易入场。
"离手"：入场后按计划执行，不让情绪改变计划。
禁止情绪驱动的改变，允许规则驱动的调整（TP达成后的SL管理）。
合规平仓仅四种：触SL · 到TP · 结构自证错误 · 周期结束平仓。

## 桌风与下注平衡
- 顺风：标准入池，对应牌型下注模式
- 逆风（标准）：按Ch5决策表——AA降级入池（KK执行）；KK条件入池（7+ + 纹理不湿润 + R:R≥2:1，三缺一弃牌）；AK/AQ弃牌
- 逆风（新手简化）：仅AA入池，KK/AK/AQ直接弃牌

## 错失甜点位处理
1. 放弃（价格已远离，无新结构形成）
2. 等新发牌（同一或其他边界出现新RE/AC信号，走完整验牌→评牌流程）
3. 延续牌（桌风不反向 + 新小平衡成型 + 小平衡内顺势Radar信号 → 轻仓入场，详见Ch5）

## 入池SOP（反恐惧防踏空）
Q1 桌成熟了吗？→ 不成熟就用上一桌（Ch3§3.2）
Q2 有发牌吗？→ 有就验牌（Ch4），没有就等
Q3 怕什么？→ 怕逆风看Ch5决策表，怕错过看Ch5延续牌，怕亏损看Ch7买定离手

## Poker Radar（辅助工具）
四种信号：uAC(向上接受) / dAC(向下接受) / uRE(向上拒绝) / dRE(向下拒绝)
评分：7+优质 / 4-6普通 / 4以下忽略。
Radar 只提高执行质量，不改变开仓条件。
